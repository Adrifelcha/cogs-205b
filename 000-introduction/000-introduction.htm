  • use branches

Example workflow

  1. git clone git@github.com:user/repo.git
  2. git -C repo/ checkout -b small-edits
  3. (make new file)
  4. git -C repo/ add .
  5. (make small edit)
  6. git -C repo/ commit -a -m "Small change"
  7. (make small edit)
  8. git -C repo/ commit -a -m "Tiny change"
  9. git push
  10. (open pull request)

Example workflow

  1. git checkout main
  2. git fetch --all --prune
  3. git rebase
  4. git -C repo/ checkout -b bugfix
  5. (make small edit)
  6. git -C repo/ commit -a -m "Fixed bug"
  7. git push
  8. (open pull request)
